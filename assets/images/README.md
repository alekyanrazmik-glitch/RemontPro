# Изображения, видео и портфолио — как загружать

Эта папка хранит все медиафайлы сайта RemontPro. Ниже — куда класть файлы,
как их называть и что обновить после загрузки.

## Структура папок

```
assets/
├── images/
│   ├── hero/             — фото для главной (баннер, обложки секций)
│   ├── portfolio/        — фото объектов портфолио
│   │   ├── object-01/
│   │   ├── object-02/
│   │   └── object-03/
│   ├── before-after/     — фото «до / после» ремонта
│   ├── team/             — фото команды/мастеров
│   ├── reviews/          — фото к отзывам клиентов
│   └── icons/            — иконки и мелкая графика
├── videos/               — видео и видео-превью
└── uploads/              — временная загрузка (черновики, файлы из админки)
```

## 1. Фото для главной

Загружайте в `assets/images/hero/`.
Сюда относятся: главный баннер (hero), обложки блоков на главной странице,
большие фоновые фотографии.

## 2. Фото объектов (портфолио)

Каждый объект — своя папка внутри `assets/images/portfolio/`:
`object-01/`, `object-02/`, `object-03/` и т.д.
При добавлении нового объекта создайте новую папку с продолжением нумерации
(`object-04/`, `object-05/`...).

В папку объекта кладите все фото этого объекта: обложку (cover) и галерею.

## 3. Как называть фотографии

- Только латиница, строчные буквы, цифры и дефис. Без пробелов, кириллицы
  и заглавных букв.
- Имя должно описывать содержимое.
- Примеры:
  - `hero-main.jpg` — главный баннер
  - `object-01-cover.jpg` — обложка объекта 01
  - `object-01-kitchen-01.jpg`, `object-01-bathroom-02.jpg` — галерея объекта
  - `before-01.jpg`, `after-01.jpg` — пара «до / после»
  - `team-ivan-petrov.jpg` — фото мастера
  - `review-anna.jpg` — фото к отзыву
- Форматы: `.jpg` / `.webp` для фото, `.png` для графики с прозрачностью,
  `.svg` для иконок.
- Перед загрузкой сжимайте фото (рекомендуемая ширина — до 1920px для hero
  и до 1280px для галереи).

## 4. Фото «до / после»

Загружайте в `assets/images/before-after/`.
Используйте парные имена с одинаковым номером: `before-01.jpg` + `after-01.jpg`.

## 5. Фото команды и отзывов

- Фото команды/мастеров — в `assets/images/team/`.
- Фото к отзывам клиентов — в `assets/images/reviews/`.

## 6. После загрузки — обновите данные

Загрузка файла в папку сама по себе ничего не покажет на сайте. После
загрузки нужно прописать пути к фото:

- **Объекты портфолио, обложки, галереи, «до/после», видео** — обновите
  `assets/data/projects.json`. Используемые поля: `cover`, `gallery`,
  `before`, `after`, `youtube`.
  Пример пути: `"cover": "assets/images/portfolio/object-01/object-01-cover.jpg"`.
- Либо внесите изменения через **админку** (`/admin`), если используете её.
- Видео и видео-превью — `assets/data/videos.json`, файлы кладите в
  `assets/videos/`.

> Папка `assets/uploads/` — временная: для черновиков и файлов, загруженных
> через админку. После того как фото распределены по нужным папкам и
> прописаны в `projects.json`, очищайте `uploads/`.
